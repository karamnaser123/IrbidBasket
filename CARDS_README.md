# صفحة الكروت - Cards Page

## الميزات المضافة

### 1. صفحة الكروت الرئيسية (Cards Page)
- **الموقع**: `lib/cards_page.dart`
- **الوصف**: صفحة لعرض جميع كروت المستخدم مع إحصائياتها
- **الميزات**:
  - عرض ملخص عام للإحصائيات
  - عرض كروت ذهبية مع إحصائياتها
  - عرض كروت فضية مع إحصائياتها
  - عرض كروت برونزية مع إحصائياتها
  - إمكانية الوصول لتفاصيل كل كارت
  - زر تحديث لإعادة تحميل البيانات

### 2. صفحة تفاصيل الكارت (Card Details Page)
- **الموقع**: `lib/card_details_page.dart`
- **الوصف**: صفحة لعرض تفاصيل كارت محدد
- **الميزات**:
  - معلومات الكارت الأساسية
  - إحصائيات مفصلة (المبيعات، الإيرادات، متوسط الطلب)
  - قائمة المبيعات الأخيرة
  - تصميم متجاوب مع نوع الكارت

### 3. نماذج البيانات (Card Models)
- **الموقع**: `lib/models/card_models.dart`
- **الوصف**: نماذج لبيانات الكروت المختلفة
- **النماذج**:
  - `CardSale`: بيانات المبيعات
  - `CardStatistics`: إحصائيات الكارت
  - `BaseCard`: الكارت الأساسي
  - `GoldCard`: الكارت الذهبي
  - `SilverCard`: الكارت الفضي
  - `BronzeCard`: الكارت البرونزي
  - `UserCardsResponse`: استجابة كروت المستخدم
  - `CardSummary`: ملخص الكروت

### 4. خدمة الكروت (Cards Service)
- **الموقع**: `lib/services/cards_service.dart`
- **الوصف**: خدمة للتعامل مع APIs الكروت
- **الوظائف**:
  - `getUserCards()`: جلب كروت المستخدم
  - `getCardDetails()`: جلب تفاصيل كارت محدد
  - `getUserCardSummary()`: جلب ملخص كروت المستخدم

## كيفية الاستخدام

### الوصول إلى صفحة الكروت
1. من صفحة QR Code، اضغط على أيقونة الكارت في الشريط العلوي
2. ستنتقل إلى صفحة الكروت الرئيسية

### عرض تفاصيل كارت
1. في صفحة الكروت، اضغط على أي كارت
2. ستنتقل إلى صفحة تفاصيل الكارت

## API Endpoints

### جلب كروت المستخدم
```
GET /api/user-cards?user_id={user_id}
Headers: Authorization: Bearer {token}
Response: {
  "status": "success",
  "data": {
    "user": {...},
    "overall_statistics": {...},
    "gold_cards": [...],
    "silver_cards": [...],
    "bronze_cards": [...]
  }
}
```

### جلب تفاصيل كارت محدد
```
GET /api/card-details?card_id={card_id}&card_type={card_type}
Headers: Authorization: Bearer {token}
Response: {
  "status": "success",
  "data": {
    "id": 1,
    "card_number": "GOLD001",
    "statistics": {...},
    "recent_sales": [...]
  }
}
```

### جلب ملخص كروت المستخدم
```
GET /api/user-card-summary?user_id={user_id}
Headers: Authorization: Bearer {token}
Response: {
  "status": "success",
  "data": {
    "gold_cards": {...},
    "silver_cards": {...},
    "bronze_cards": {...},
    "totals": {...}
  }
}
```

## التصميم

### الألوان المستخدمة
- **الخلفية**: تدرج من الأسود إلى الأزرق الداكن
- **اللون الرئيسي**: `#e94560` (أحمر غامق)
- **كروت ذهبية**: `Colors.amber`
- **كروت فضية**: `Colors.grey`
- **كروت برونزية**: `Colors.orange`

### الميزات التصميمية
- تصميم عصري وخفيف
- ألوان مميزة لكل نوع كارت
- بطاقات شفافة مع حدود ملونة
- أيقونات واضحة لكل نوع
- إحصائيات منظمة ومقروءة

## أنواع الكروت

### 1. الكروت الذهبية (Gold Cards)
- أعلى مستوى في النظام
- تحتوي على محفظة رقمية
- يمكن أن تحتوي على كروت فضية وبرونزية
- إحصائيات شاملة

### 2. الكروت الفضية (Silver Cards)
- مستوى متوسط
- مرتبطة بكارت ذهبي
- يمكن أن تحتوي على كروت برونزية
- إحصائيات محددة

### 3. الكروت البرونزية (Bronze Cards)
- المستوى الأساسي
- مرتبطة بكارت فضي
- إحصائيات بسيطة

## الإحصائيات المعروضة

### الإحصائيات العامة
- إجمالي عدد الكروت
- إجمالي المبيعات
- إجمالي الإيرادات

### إحصائيات الكارت
- إجمالي المبيعات
- إجمالي الإيرادات
- متوسط قيمة الطلب
- رصيد المحفظة (للذهبية)
- عدد الكروت الفرعية
